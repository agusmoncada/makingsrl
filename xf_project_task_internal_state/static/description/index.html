Internal State for Tasks

The module introduces internal states (new, open, suspended, done) for tasks and task stages to facilitate the easy determination of task statuses.

In Odoo, there are usually many stages, making it technically difficult to determine whether a task is in progress, completed, or not yet started. Our module introduces the concept of internal states to enable easy determination of task status and to group tasks based on this status.
