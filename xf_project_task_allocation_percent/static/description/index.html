Project Task Allocation Percent

The module helps to control resource allocation percentage. Each user can set Assignment Units for task from 0 to 100 and the system will control Percent Allocation per Employee.

Key Features:

  • User can set own assignment units for each task.
  • If percent allocation is greater than maximum allowed percentage, the system prevents starting new task.
  • Restrict starting multiple parallel tasks

How it works:

User can set own assignment units for each task

The project module administrator can set maximum allowed percent allocation per employee:

When maximum percent allocation is set and user try to start new task, the system will check total amount of a resource's capacity allocated to tasks.

If percent allocation is greater than maximum allowed percentage, the system prevents starting new task.

As you can see, with this module, you can control employees workload. Also, Odoo will restrict starting multiple parallel tasks and prevent overallocation of resources (employees).
